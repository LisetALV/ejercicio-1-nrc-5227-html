Atributo ID

El atributo global id define un identificador único (ID), para una etiqueta, el cual no debe repetirse en todo el documento HTML.

Su propósito es identificar el elemento, al vincularlo, en scripts u hojas de estilo (con CSS)

El valor de este atributo es una cadena de caracteres y no debe contener espacios en blanco.
